CAPTCHA solving AI

This project contains a keras model that can train on and learn how to solve CAPTCHAs (original) with a high degree of accuracy within a small number of generations. It uses keras with a tensorflow backend (made by google) and is a simple convolutional classification model.

Machine Learning Racecars

This project utilises a ML library called Tensorflow to create small car-like critters that are able to navigate a racetrack. It does this by using a genetic evolution and testing each critter befor seeing if they are allowed to pass down their genes or not.

Wireworld Circuits

This project is a crude imitation of electrical circuts that uses and gates, generators and wires to transmit electrical signals made up of singal electrons. In this example the posibilities are fairly limited but in more complex simulations entire computers can be constructed with wires alone.

Flow Chart

In this project 2d perlin noise is projected onto a plane and converted from a height map to a vector map. This will then direct particles who will leave trails, thus creating river-like shapes.

Polynomial Regression

This project utilises a ML library called Tensorflow to fit a polynomial line with a given complexity to a set of points. It does this by training a neural network with a linear regression algorithm.

Smart Rockets

This projects uses a custom made machine learning algorithm that trains small rockets to avoid a obstacle and reach a target. It does this by giving each rocket a list of instructions and creating imperfect copies of the best rockets.

Raycasting

This small project simulates lighting by sending our rays of 'light' that collide with boundaries. In a more sophisticated simulation they could then bounce with a fraction of their starting energy but that would be to complex to run in the browser.
