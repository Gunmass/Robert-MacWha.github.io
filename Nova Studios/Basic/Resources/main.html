Unity Documentation

The Unity Documentations is a great place to look if you don't fully understand how some of Unity's functions works.

BlackthornProd

BlackthornProd is a fantastic Youtuber that produces quality Unity tutorials on a semi-weekly basis.

Brackeys

Brackeys is another great Youtuber that produces videos on the more visual side of Unity, teaching about rendering and shaders.

Javascript

The Coding train

The Coding train is one of the best Youtube chanles to go to if you are learning Javascript. I would suggest just watching some of his videos any time you don't have any motivation.

W3-School

W3-School is a great website to go to if you need help understanding the uses of functions in Javascript.

P5.JS Website

The P5.JS website has plenty on information on the usage of both P5.js and many of it's addons. It also showes you how to make simple projects.
