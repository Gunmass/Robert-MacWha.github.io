Our Message

To give everyone, from a young age, the ability to access quality tutorials that can provide essential abilities.

Scratch

Scratch is a block-based visual programing language created by MIT in 2003. It was created primarily for younger children to help them learn basic computer programing concepts in a fun, simple way.

Unity

Unity is a very flexible game engine developed by Unity Technologies. It allowes users to created games in both 2d and 3d space and offers many assets and tools to help with game development.

Javascript

Javascript is a front end programing language used in website development, databases and much more. Unlike Scratch or Unity, Javascript isn't used in game development, but instead in creating interactive websites
